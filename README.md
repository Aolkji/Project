This is the CISC 3160 Programming Langauges project.
The project implements a simple interpreter using a tokenizer,parser,and interpreter
Tokenizer: Convert inputs into tokens
Parser: Evaluates syntax and expression
Interpreter: Executes assignments and prints results

How to use:
To run project - python3 interpreter.py
When entering inputs inorder to recieve output must type run then press enter
When finished testing to quit you must type quit then press enter
